## Engineering Signal Processing Lab

A project-based portfolio covering signal processing, control systems, system identification, modal analysis and structural health monitoring.

## Topics include

- Stage 1: Synthetic Vibration Signals
  - Generation and analysis of synthetic vibration signals to investigate sampling, aliasing, noise, frequency resolution, spectral leakage, and windowing.
- FFT
- Spectral Analysis
- Digital Filters
- STFT
- Wavelets
- Random Signals
- Modal Identification
- Kalman Filtering
- State Estimation
- PID Control
- State-Space Control
- Structural Health Monitoring
- Machine Learning Applications
